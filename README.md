# Windows Troubleshooter

A satirical WinForms app that perfectly emulates the Windows troubleshooting experience.

## What it does

1. Opens a window titled **"Windows Network Diagnostics"**
2. Cycles through impressive-sounding status messages
3. Animates a progress bar
4. After 26 seconds of theatrical scanning, reports: **"No issues found"**

That is the entire user experience. Nothing is diagnosed. Nothing is fixed.

## Why it exists

Because in a lifetime of using Windows, the built-in troubleshooting tools have never - not once - actually helped me.

## Tech

- C# / .NET Framework 4.8
- WinForms
- Zero functionality, maximum authenticity

## Usage

Build and run. The app doesn't do actually anything. Just like the real thing.
